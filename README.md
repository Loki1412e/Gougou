# Gougou - Projet

Projet en C sous windows utilisant la librairie `CSFML` version `2.6.1`, `gcc` ainsi que `make`.

## Setup et Utilisation :

```bash
git clone git@github.com:Loki1412e/Gougou.git
```

```bash
C:/path/Gougou> make
C:/path/Gougou> .\gougou.exe
```
